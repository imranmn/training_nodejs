## what is node.js?
- Node.js adalah runtime environment utk run javascrip outside browser
- Express.js adalah framework untuk back end server dan API
- PostgreSQL open-source database
- vue.js adalah framework untuk front end (UI) dari community, sama macam angular dari google, reactjs dari fb

## why node.js? 
- lebih laju, memerlukan kurang server, kurang lines of code, boleh uruskan volumn data yang berubah dengan cepat, boleh uruskan banyak I/O dalam realtime, single programming langguage
- Contoh: Netflix, linkedIn, PayPal, Uber

NPM - node package manager / composer / dependencies management
NPX - executable program
